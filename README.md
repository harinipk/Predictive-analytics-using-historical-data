Predictive Analytics Using Historical Data

Project Overview

This project uses historical Superstore sales data to build a predictive analytics model for forecasting sales trends. The dataset was cleaned, preprocessed, analyzed, and visualized before training a Linear Regression model.

Objectives

- Clean and preprocess historical sales data.
- Analyze sales trends over time.
- Build a Linear Regression model for prediction.
- Evaluate model performance.
- Visualize actual and predicted sales.

Tools & Technologies

- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

Dataset

- SuperStore_Sales_Dataset.xlsx

Project Workflow

1. Loaded the historical sales dataset.
2. Cleaned and preprocessed the data.
3. Converted the Order Date column into datetime format.
4. Removed unnecessary columns and handled missing values.
5. Visualized sales trends.
6. Built a Linear Regression model.
7. Predicted sales values.
8. Evaluated the model using Mean Absolute Error (MAE) and R² Score.
9. Saved prediction results.

Model Evaluation

- Mean Absolute Error (MAE): 232.04
- R² Score: 0.0127
